Дамп базы данных находится в файле: news.sql.
Для тестирования в БД созданы пользователи: логин - user1, пароль - password1
логин - user2, пароль - password2
логин - user3, пароль - password3

Фото пользователей и фото новостей хранятся не в самой БД, а на диске. Предполагается,
что это будет диск сервера. В самой БД хранится только имя файла с фотографией. Поменять
место хранения фото можно константой upload.path из файла свойств.
Файлы с фотографиями находся в корне проекта, в папке uploads
